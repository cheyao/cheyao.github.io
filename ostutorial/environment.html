We need to setup a environment to begin developing our OS, here I am on macOS, so there might be errors with the toolchain in other OSes.

If you are using windows, I strongly recommend you to switch to linux or get a macbook.

On macOS, we need to install Homebrew as our package manager by pasting /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" into your terminal.

Nasm

For the assembler, we will use nasm as I am the most familiar with it, install it with your package manager, on macos it is like:

brew install nasm

If this doesn't work or your distro doesn't have a package manager, download nasm from here.

Gcc toolchain

For the C compiler, we will use x86_64-elf-gcc and x86_64-elf-binutils, install it with your package manager, on macos it is like:

brew install x86_64-elf-gcc x86_64-elf-binutils

If you don't have a package manager or it doesn't work you can compile it by your self, following this article, just change export TARGET=i686-elf to export TARGET=x86_64-elf.

Note: x86_64-elf-gcc/binutils is not the same as i686-elf-gcc/binutils or your OS's native gcc/binutils.

For the emulator, we will use qemu, install it with your package manager:
