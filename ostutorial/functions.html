In the last lesson, we made our bootsector print out Hello on the screen, but repeating three lines of code for every character isn't really handy, isn't it?

Labels

Labels are used to declare an address, you declare them by doing: your_label_name: and use them by doing: your_label_name, at compile time, they we be changed to the address of where the label is declared.

The stack

The stack is a useful way to store variables, we can store a value using the push instruction and get one using the pop instruction. The newest value's position is defines by esp and the start of the stack by ebp.

Imagine that you want to have an instruction that prints A onto the screen, but we would like to preserve the eax register, so we will do it like this: 

puta:
    push eax
    mov ah, 0x0E
    mov al, 'A'
    int 0x10
    pop eax
    ret ; ret is an instruction to get the last value on the stack, and return there

Now if we want to use the function, we will use call puta, the instruction pushes the current position onto the stack and jumps to puta.

Creating a putchar function

To create a putchar function we can do it like this: 

start:
    mov al, 'H'
    call putchar
    mov al, 'i'
    call putchar
    jmp $

putchar:
    push eax
    mov ah, 0x0e
    int 0x10
    pop eax
    ret

Now we have a little function that prints a character, but it still isn't really useful, so lets make a puts function.

Puts function

To make a puts function, we still have to learn a few things.

1. defining a string

Strings, is just a pointer to a group of characters, so to define string1, we can use db (define byte, byte = char), like this:

string1:
    db "Technoblade", 0

Notice how I also defined a 0 in the end of the code. 0 is commonly used as a string end character, which means (as it's name), the end of a string

2. conditions

To do a simple statement like this:

if (eax == 0) {
    eax++;
}

We would need to use the cmp command, like this:

compare:
    cmp eax, 0 ; Can be understood as get the relationships between the value in eax and 0
    jne end ; Jump if not equal
    add eax, 1 ; This gets executed only if eax is equal to zero>
end:

3. Implementation

This will be our boot/boosect.asm now:

org 0x7C00
bits 16 ; Tell nasm we are in 16 bit mode

 
start: 
    mov di, string ; Store the position of our string into edi
    call puts ; Call the function
    jmp $ 
 
puts: ; Our print function
    push eax ; Store the registers
    push di 
    mov ah, 0x0E 
.loop: 
    mov al, [di] ; Store the stuff in memory position edi to al
    cmp al, 0 
    je end ; If al is equal to 0, goto end
    int 0x10
    inc di 
    jmp .loop
end: 
    pop di 
    pop eax 
    ret 
 
string: 
    db "Hello world!", 0 
 
times 510-($-$$) db 0 ; Padding and magic number
    dw 0xaa55

Now re-compile the bootsector and run it in qemu, you will see that we now have Hello world! printed on the screen like this!
