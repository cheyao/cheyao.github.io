In the last chapter, we made a bootable disk, but it doesn't do anything. Now lets make it print "Hello" onto the screen.

Interrupts

When the computer boots, the BIOS initializes some standard interrupts (Here is a full list of interrupts).

int 0x10 is the interrupt for video services, the ah register represents the function code, what we want to use, is int 0x10, ah = 0x0E which is teletype output

Implementing

In the interrupt function, al is containing the ascii code of the letter, we can get the ascii by putting a character inside apostrophes. Now this is our boot/bootsect.asm:

org 0x7C00
mov ah, 0x0e

mov al, 'H'
int 0x10
mov al, 'e'
int 0x10
mov al, 'l'
int 0x10
mov al, 'l'
int 0x10
mov al, 'o'
int 0x10

jmp $

times 510 - ($ - $$) db 0 ; Fill with 510 zeros minus the size of the previous code
dw 0xaa55 ; The magic number

The mov instruction moves the value in the second argument to the register in the first register.

Now when we recompile it and rerun qemu, we can see that Hello got printed, like this.
